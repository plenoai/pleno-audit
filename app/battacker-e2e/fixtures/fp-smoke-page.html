FP Smoke Test Legitimate Web Patterns

This page exercises common web APIs that legitimate websites use daily. None of these patterns should produce a security alert.

Home About Docs

1. SPA Hydration

localStorage.getItem × 30
Threshold: 50 calls / 3 s. Next.js/Nuxt hydration reads state, theme, locale, feature flags, etc.

2. Analytics / GTM

sendBeacon < 1 KB + 3 cookie reads
Beacon threshold: > 1 KB. Cookie threshold: ≥ 5 reads / 3 s. We stay below both.

3. Media Player

AudioContext + <video> fullscreen
AudioContext hook fires unconditionally (known FP source). Video fullscreen is whitelisted.

4. Real-time Communication

RTCPeerConnection + addTrack
RTCPeerConnection hook fires unconditionally (known FP source). Used by video call apps.

5. PWA

ServiceWorker.register()
Hook fires unconditionally (known FP source). Will fail on non-HTTPS — that is expected.

6. CDN Resources

Static dns-prefetch / preconnect links
Only dynamically injected links trigger the MutationObserver. Static <link> in <head> is safe.

7. CSS-in-JS

Dynamic <style> injection (no input[value] + background-image)
CSS keylogging detection requires both input[value and background-image: url( patterns.

8. Normal HTML

Static forms, images, anchors with id attributes
DOM clobbering detection only watches dynamically added nodes. Static elements are safe.

9. Worker Computation

Web Worker via blob: URL
Worker hook fires unconditionally (known FP source). Used for off-main-thread computation.

10. JSON.parse

JSON.parse for data deserialization
JSON.parse is NOT hooked (unlike eval/Function). Completely safe.

11. execCommand('copy')

Copy-to-clipboard button
execCommand hook fires unconditionally (known FP source). Used by every "copy" button on the web.

12. Blob Download

URL.createObjectURL for file download
Suspicious download hook fires unconditionally (known FP source). Used for client-side file generation.

13. querySelectorAll

~50 calls (threshold: 300 / 5 s)
SPA frameworks routinely call querySelectorAll. 50 calls is well within safe range.

14. Cookie Reads

3 reads (threshold: 5 / 3 s)
Normal page load reads session, consent, and locale cookies.
